# Calculadora
Desarrollar una calculadora web capaz de realizar operaciones básicas y avanzadas, procesar listas de números mediante arrays y mostrar resultados dinámicamente en el DOM. Debe incluir validación de datos y gestión de errores para evitar operaciones no válidas y mejorar la robustez de la aplicación.


# Fase 1: Operaciones con dos números
Desarrolla una interfaz funcional que permita realizar las cuatro operaciones aritméticas básicas (+, -, *, /) utilizando dos valores de entrada. El sistema debe capturar los datos introducidos por los usuarios, procesar el cálculo mediante funciones modulares y actualizar el contenido de un elemento específico del DOM para mostrar el resultado final.

# Fase 2: Procesamiento de colecciones (Sumatorio)
Implementa una funcionalidad adicional que permita recibir una lista indeterminada de números. La aplicación debe ser capaz de almacenar estos valores en una estructura de array y recorrerlos mediante métodos de iteración para devolver la suma total de todos los elementos de la colección.

# Fase 3: Cálculos avanzados y control de robustez
Amplía las capacidades de la calculadora integrando operaciones de potencias, raíces cuadradas y el uso de la constante "pi" mediante el objeto global Math. En esta fase es obligatorio implementar un sistema de gestión de errores que intercepte entradas no numéricas u operaciones matemáticas indefinidas (como la división por cero), informando al usuario del error sin detener la ejecución del programa.
